About the data

Grid-n-berg

Grid-n-berg is a topography-aware pedestrian routing dataset for New York City. It starts from an existing pedestrian network, assigns local elevation to network vertices, and compares routes using distance-only, comfort, and accessibility-sensitive cost profiles.

Network
NYC pedestrian network geometry from Sevtsuk et al.
Elevation
NYC Planimetric Database elevation points.
Method
Average nearby elevation points within 50 meters.
Costs
Directional slope penalties stored on each segment.

Slope evaluation

Routing scenarios

The accessible profile is a slope-aware scenario, not an ADA compliance label. The dataset does not measure cross slope, curb ramps, sidewalk width, surface condition, obstacles, legal access, or temporary conditions.
